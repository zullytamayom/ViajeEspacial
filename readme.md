# 🚀 Starbound - Turismo Interespacial

**Starbound** es una landing page interactiva desarrollada como ejercicio de hackathon, enfocada en ofrecer experiencias de turismo intergaláctico de forma visual, dinámica y atractiva.

## 🌌 Características principales

* 🛰️ Catálogo interactivo de destinos espaciales
* 🛒 Sistema de carrito con almacenamiento en **localStorage**
* 🚀 Animaciones dinámicas (transiciones, partículas, slider)
* 🧾 Confirmación de compra mediante modal
* 🌠 Integración con la API de la NASA (Astronomy Picture of the Day)

## 🔗 API utilizada

Se consume la API pública de la NASA:

* **APOD (Astronomy Picture of the Day)**
  https://api.nasa.gov/


## 🛠️ Tecnologías

* HTML5
* CSS3 (animaciones y diseño responsivo)
* JavaScript (DOM, eventos, localStorage)
* Bootstrap 5
